# 💸 FinSight AI – Finance Dashboard

A full-stack finance dashboard that tracks transactions, analyzes spending, and provides smart insights.

---

## 🚀 Features

- 📊 Interactive dashboard UI
- 💰 Transaction tracking (income & expenses)
- 🔗 Backend API integration
- 📈 Data visualization (charts & analytics)
- ⚡ Real-time data rendering

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Recharts

### Backend
- Node.js
- Express
